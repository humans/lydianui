# Combobox

## `<Combobox>`

### Props

| Name       | Type      | Default     | Description                            |
| ---------- | --------- | ----------- | -------------------------------------- |
| `as`       | `string`  | `div`       | Tag to render the component.           |
| `value`    | `any`     | `undefined` |                                        |
| `multiple` | `boolean` | `false`     | Saves the selected values in an array. |
